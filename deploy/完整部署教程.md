# AI数字人陪伴平台 - 从零开始完整部署教程

## 前言

本教程将手把手教您如何从零开始部署AI数字人陪伴平台到生产环境。

## 第一步：准备工作

### 1.1 购买云服务器

推荐平台：
- **阿里云**：https://www.aliyun.com
- **腾讯云**：https://cloud.tencent.com

推荐配置：
```
CPU: 2核
内存: 4GB
带宽: 5Mbps
系统盘: 50GB SSD
系统: Ubuntu 20.04 LTS
价格: 约 300-500元/年
```

购买后您会得到：
- 服务器公网IP（如：123.45.67.89）
- root用户名和密码

### 1.2 购买域名

在阿里云或腾讯云购买域名：
- 推荐 `.com` 后缀（如：`aicompanion.com`）
- 价格：约 50-100元/年

### 1.3 域名备案（国内服务器必须）

如果您购买的是中国大陆服务器，必须进行ICP备案：

1. 登录阿里云/腾讯云控制台
2. 找到"备案"入口
3. 提交备案申请：
   - 个人备案：需要身份证正反面照片
   - 企业备案：需要营业执照
4. 填写网站信息
5. 等待审核（7-20个工作日）

**注意**：备案期间网站不能访问

## 第二步：连接服务器

### 2.1 Windows用户

下载安装 **PuTTY**：
1. 官网下载：https://www.putty.org/
2. 打开 PuTTY
3. 在 Host Name 输入：`root@您的服务器IP`
4. 点击 Open
5. 输入密码（输入时不显示，直接回车）

### 2.2 Mac/Linux用户

打开终端，执行：
```bash
ssh root@您的服务器IP
# 例如：ssh root@123.45.67.89
```

输入密码后回车

### 2.3 首次连接后的必要操作

```bash
# 更新系统软件
apt update && apt upgrade -y

# 设置时区为中国
 timedatectl set-timezone Asia/Shanghai

# 查看当前时间
date
```

## 第三步：安装环境（自动脚本）

### 3.1 上传安装脚本

在您的电脑上，将 `deploy/install-all.sh` 上传到服务器。

使用 WinSCP 或 FileZilla：
1. 下载安装 WinSCP
2. 连接服务器（IP、root、密码）
3. 将 `install-all.sh` 拖到服务器根目录

### 3.2 运行安装脚本

在服务器上执行：

```bash
# 进入根目录
cd /

# 给脚本执行权限
chmod +x install-all.sh

# 运行脚本
bash install-all.sh
```

脚本会自动安装：
- Node.js 18
- PM2（进程管理器）
- MongoDB 6.0
- Nginx
- Git
- Certbot（SSL证书）

安装过程约 5-10 分钟。

## 第四步：上传项目代码

### 4.1 打包本地代码

在您的电脑上：

**Windows:**
1. 选中 `backend`、`frontend`、`deploy`、`README.md`
2. 右键 → 发送到 → 压缩文件夹
3. 得到 `ai-companion.zip`

**Mac:**
```bash
zip -r ai-companion.zip backend frontend deploy README.md
```

### 4.2 上传到服务器

使用 WinSCP/FileZilla：
1. 连接服务器
2. 进入 `/var/www/` 目录
3. 创建 `ai-companion` 文件夹
4. 将 `ai-companion.zip` 上传到该文件夹

### 4.3 解压代码

在服务器上执行：

```bash
# 进入目录
cd /var/www/ai-companion

# 安装解压工具
apt install -y unzip

# 解压
unzip ai-companion.zip

# 删除压缩包
rm ai-companion.zip
```

## 第五步：配置后端

### 5.1 安装后端依赖

```bash
cd /var/www/ai-companion/backend

# 安装依赖
npm install

# 创建日志目录
mkdir -p logs
```

### 5.2 配置环境变量

```bash
# 复制配置文件
cp .env.production .env

# 编辑配置文件
nano .env
```

按实际情况修改以下内容：

```env
# 数据库（保持默认即可）
MONGODB_URI=mongodb://localhost:27017/ai_companion

# JWT密钥（设置一个复杂的密码）
JWT_SECRET=your_very_strong_secret_key_123456789

# 微信小程序（必须填写）
WECHAT_APPID=wx_您的小程序appid
WECHAT_SECRET=您的小程序secret

# 微信支付（必须填写）
WECHAT_MCH_ID=您的商户号
WECHAT_API_KEY=您的API密钥
WECHAT_NOTIFY_URL=https://您的域名/api/payment/notify

# AI大模型（选择其一）
# OpenAI
AI_API_KEY=sk-your_openai_api_key

# 或文心一言
# WENXIN_API_KEY=您的文心一言key
# WENXIN_SECRET=您的文心一言secret

# 短信服务（阿里云）
SMS_ACCESS_KEY=您的阿里云access_key
SMS_SECRET_KEY=您的阿里云secret_key
SMS_SIGN_NAME=您的短信签名
SMS_TEMPLATE_CODE=SMS_您的模板代码
```

保存：按 `Ctrl+O`，回车，然后 `Ctrl+X` 退出

### 5.3 启动后端服务

```bash
# 使用PM2启动
npm run pm2:start

# 查看状态
pm2 status

# 查看日志
pm2 logs ai-companion-backend
```

看到 `online` 表示启动成功！

## 第六步：配置前端

### 6.1 修改API地址

在您的电脑上修改 `frontend/src/utils/request.ts`：

```typescript
// 修改这一行
const BASE_URL = 'https://您的域名/api'
// 例如：const BASE_URL = 'https://aicompanion.com/api'
```

### 6.2 构建前端

在您的电脑上：

```bash
cd frontend

# 安装依赖
npm install

# 构建生产版本
npm run build
```

构建完成后，会生成 `dist` 文件夹。

### 6.3 上传前端代码

使用 WinSCP：
1. 将 `frontend/dist` 文件夹上传到服务器
2. 放到 `/var/www/ai-companion/frontend/dist`

## 第七步：配置Nginx

### 7.1 修改Nginx配置

编辑 `deploy/nginx.conf`，将 `your-domain.com` 替换为您的真实域名：

```bash
# 在服务器上执行
cd /var/www/ai-companion/deploy

# 替换域名
sed -i 's/your-domain.com/您的真实域名/g' nginx.conf

# 复制到Nginx配置目录
cp nginx.conf /etc/nginx/sites-available/ai-companion

# 创建软链接
ln -sf /etc/nginx/sites-available/ai-companion /etc/nginx/sites-enabled/

# 删除默认配置
rm -f /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 重启Nginx
systemctl restart nginx
```

### 7.2 配置SSL证书（HTTPS）

```bash
# 申请证书
certbot --nginx -d 您的域名 -d www.您的域名

# 按提示操作：
# 1. 输入邮箱
# 2. 同意协议
# 3. 选择是否接收邮件
```

证书会自动续期。

## 第八步：微信小程序配置

### 8.1 注册小程序账号

1. 访问 https://mp.weixin.qq.com
2. 点击"立即注册"
3. 选择"小程序"
4. 填写企业信息，完成认证（300元/年）

### 8.2 获取AppID和Secret

1. 登录小程序后台
2. 左侧菜单 → 开发 → 开发管理
3. 找到 AppID(小程序ID)
4. 生成 AppSecret(小程序密钥)

### 8.3 配置服务器域名

1. 小程序后台 → 开发 → 开发管理 → 服务器域名
2. 添加以下域名：
   - request合法域名：`https://您的域名`
   - uploadFile合法域名：`https://您的域名`
   - downloadFile合法域名：`https://您的域名`

### 8.4 修改小程序代码

在您的电脑上修改 `frontend/src/manifest.json`：

```json
{
  "name": "AI数字人陪伴",
  "appid": "wx_您的小程序appid",
  ...
}
```

### 8.5 上传代码

1. 下载微信开发者工具
2. 导入项目
3. 点击"上传"
4. 填写版本号
5. 提交审核

## 常见问题

### Q1: 无法连接服务器
- 检查安全组是否开放22端口
- 检查密码是否正确

### Q2: 网站无法访问
- 检查Nginx是否运行：`systemctl status nginx`
- 检查后端是否运行：`pm2 status`
- 检查域名解析是否正确

### Q3: 小程序无法连接
- 检查服务器域名是否配置正确
- 检查HTTPS证书是否有效
- 检查API地址是否为HTTPS

### Q4: 支付功能异常
- 检查微信支付配置
- 检查notify_url是否正确
- 查看支付日志
