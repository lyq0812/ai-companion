# 免费内网穿透方案（临时演示用）

## 方案：使用 ngrok（免费）

### 1. 注册 ngrok
- 访问 https://ngrok.com
- 用 GitHub/Google 账号登录
- 获取 Authtoken

### 2. 下载安装
```bash
# Windows
choco install ngrok

# 或手动下载解压
```

### 3. 配置
```bash
ngrok config add-authtoken 您的token
```

### 4. 启动穿透

**先确保本地服务在运行：**
- 后端：http://localhost:8080
- 前端：http://localhost:3000

**然后启动 ngrok：**

暴露后端：
```bash
ngrok http 8080
```

会得到一个公网地址，如：`https://abc123.ngrok.io`

### 5. 修改前端配置

修改 `frontend/src/utils/request.ts`：
```typescript
const BASE_URL = 'https://abc123.ngrok.io/api'
```

然后重新构建前端。

### 6. 暴露前端
```bash
ngrok http 3000
```

得到前端公网地址，可以分享给朋友访问！

**注意：**
- 免费版域名每次重启都会变
- 适合临时演示，不适合正式使用
- 每月有流量限制

## 其他免费内网穿透工具

### 1. Cloudflare Tunnel（免费，推荐）
```bash
# 安装 cloudflared
# Windows下载: https://github.com/cloudflare/cloudflared/releases

# 登录
cloudflared tunnel login

# 创建隧道
cloudflared tunnel create my-tunnel

# 运行
cloudflared tunnel run my-tunnel
```

### 2. LocalTunnel（最简单）
```bash
# 安装
npm install -g localtunnel

# 使用
lt --port 8080
```

### 3. Pagekite
```bash
# 安装
pip install pagekite

# 使用
pagekite 8080 yourname.pagekite.me
```

## 快速操作步骤

### 第一步：准备代码
```bash
# 1. 创建 GitHub 仓库
# 2. 推送代码
git init
git add .
git commit -m "initial"
git remote add origin https://github.com/您的用户名/ai-companion.git
git push -u origin main
```

### 第二步：部署后端到 Render
1. 登录 https://render.com
2. 点击 "New +" → "Web Service"
3. 连接 GitHub 仓库
4. 配置：
   - Name: `ai-companion-backend`
   - Environment: `Node`
   - Build Command: `cd backend && npm install`
   - Start Command: `cd backend && npm start`
   - Plan: `Free`
5. 添加环境变量（在 Dashboard → Environment）
6. 点击 Create

等待部署完成，获得后端地址：`https://ai-companion-backend.onrender.com`

### 第三步：部署前端到 Vercel
1. 登录 https://vercel.com
2. 点击 "Add New Project"
3. 导入 GitHub 仓库
4. 配置：
   - Framework Preset: `Other`
   - Build Command: `cd frontend && npm install && npm run build`
   - Output Directory: `frontend/dist`
5. 添加环境变量：
   - `VITE_API_URL`: `https://ai-companion-backend.onrender.com/api`
6. 点击 Deploy

等待部署完成，获得前端地址：`https://ai-companion.vercel.app`

### 第四步：配置跨域
修改 `backend/src/app.js`，添加 Vercel 域名到白名单：
```javascript
app.use(cors({
  origin: ['https://ai-companion.vercel.app', 'http://localhost:3000']
}));
```

### 第五步：测试
访问 `https://ai-companion.vercel.app`，测试功能是否正常！
