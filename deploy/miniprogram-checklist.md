# 微信小程序上线检查清单

## 1. 账号准备
- [ ] 注册微信小程序账号（企业主体）
- [ ] 完成企业认证（300元/年）
- [ ] 开通微信支付（如需支付功能）

## 2. 域名配置
- [ ] 购买域名并完成备案
- [ ] 配置服务器域名（request合法域名）
- [ ] 配置uploadFile合法域名
- [ ] 配置downloadFile合法域名
- [ ] 配置socket合法域名（如需要WebSocket）

## 3. 小程序后台配置
- [ ] 设置服务器域名
- [ ] 设置业务域名（web-view组件）
- [ ] 配置IP白名单
- [ ] 获取AppID和AppSecret

## 4. 代码配置
- [ ] 修改 manifest.json 中的 appid
- [ ] 修改 request.ts 中的 BASE_URL 为生产环境地址
- [ ] 确保所有接口使用HTTPS
- [ ] 确保所有图片资源使用HTTPS

## 5. 隐私协议
- [ ] 编写隐私政策
- [ ] 编写用户协议
- [ ] 在小程序后台配置隐私协议
- [ ] 在代码中添加隐私协议弹窗

## 6. 内容安全
- [ ] 接入内容安全API（文本、图片审核）
- [ ] 配置敏感词过滤
- [ ] 确保无违规内容

## 7. 审核材料
- [ ] 准备软件著作权证书（如有）
- [ ] 准备ICP备案截图
- [ ] 准备特殊行业资质（如涉及）
- [ ] 准备测试账号

## 8. 提审前检查
- [ ] 功能完整测试
- [ ] 性能优化（首屏加载<2s）
- [ ] 适配测试（各机型）
- [ ] 用户体验优化
- [ ] 错误处理完善

## 9. 提审
- [ ] 填写版本信息
- [ ] 填写功能介绍
- [ ] 上传截图和预览视频
- [ ] 提交审核

## 10. 上线后
- [ ] 配置告警通知
- [ ] 配置数据分析
- [ ] 准备客服回复话术
- [ ] 制定运营计划
